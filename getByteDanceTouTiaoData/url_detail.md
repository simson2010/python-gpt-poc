## About '_signature' 
It's generated by function from `window.byted_acrawler.sign()` with parameter an object: 
```JSON
{url: "https://www.toutiao.com/api/pc/list/feed?channel_id=3189398999&max_behot_time=1695483895&offset=0&category=pc_profile_channel&client_extra_params=%7B%22short_video_item%22:%22filter%22%7D&aid=24&app_name=toutiao_web"}
```
